누구나 직접 프로젝트 제작이 가능하다

누구나 프로젝트 제작이 가능하기때문에 
펀딩상품에 신뢰성이 떨어진다.

와디즈
장점: 누구나 직접 프로젝트 제작이 가능하다.
펀딩게시판에 카테고리 기능이 있다.
단점: 펀딩에 광고가 껴있어서

펀딩상품에 신뢰성이 떨어진다.


프리모아
장점:제작사의 주요 작업 분류와 보유기술, 
그리고 활동요약과 소비자에게 전하고자하는 메세지를 볼수있어 
소비자가 직접 선택할수있어 좋다
단점: 전문적인 구인사이트 느낌이 많아 
펀딩제작사의 어필은 할수 있으나 
내가 마치 기업의 대표로써 회사에서 일할 사람을 찾는 느낌이다.



6.개발 목표 및 설계의 주안점
크라우드는 일반 소비자가 접근하기에 
투자형 상품은 진입장벽이 너무 높기때문에
투자형 상품은 없에고 직접 실물이 있는 펀딩만 진행 하기로 하였으며
프로젝트 제작은 구루밍이 승인한 회원만이 가능하게 하여 프로젝트의 신뢰성을 높였으며
크라우드는 깔끔한 UI, UX로 전하고자하는 메세지를 한눈에 쉽게 전달한다는
장점이 있기때문에 UI 및 UX 디자인을 크라우드와 유사하게 제작하였다.
프리모아는 제작사(펀더)의 주요 작업 분류와 보유기술, 
그리고 활동요약과 소비자에게 전하고자하는 메세지를 볼수있어 
소비자가 직접 선택할수있어 좋다는 장점이 있기때문에
소비자가 직접 펀더의 대해 카테고리별로 분류 및 검색을 할 수있는 
펀더 게시판을 추가하였다.

8.주요 기능 및 기능 상세
(1) 회원 정보 관련 기능
=>회원가입, 로그인/로그아웃, 비밀번호 찾기, 회원정보 수정
(2) 관리자페이지 관련 기능
=> 신고된 펀더 로그인 제한기능/ 공지사항 등록/ 이벤트 등록/ 펀더 회원가입 승인
(3) 공지사항 게시판 관련 기능
=> 공지사항 조회, 등록, 수정, 삭제 기능
(4) 이벤트 게시판 관련 기능
=> 이벤트 조회, 등록, 수정, 삭제 기능
(5) 펀딩 페이지 관련 기능
=> 펀딩  등록, 수정, 기능 , 결재 기능 , 좋아요 기능
(6) 펀더 소개 페이지 관련 기능
=> 펀더 소개 등록, 조회, 수정, 삭제 기능, 좋아요 기능
(7) 마이 페이지 관련 기능
=> 일반회원 : 펀딩한 목록 조회 기능 
=> 펀더회원 : 진행중인 펀딩 조회 기능 / 펀딩 참여인원 조회기능
