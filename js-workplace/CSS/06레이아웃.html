테두리 스타일

border-style : 테두리 스타일 지정

selector {border-style : none | hidden | dashed | dotted | double | groove | inset | outset | ridge | solid}

border-width : 테두리 두께

selector {border-[위치/생략]-width : thin | medium | thick | 숫자}

border-color : 테두리 색상

selector {border-[위치/생략]-color : 색상값}

border : 두께, 스타일, 색상을 한번에 지정

selector {border-[위치/생략] : 두께 스타일 색상값}

border-redius : 박스 모서리를 둥글게하는 속성

selector {border-[위치/생략]-radius : 숫자}
위치 = > top-left / top-right / bottom-left / bottom-right
