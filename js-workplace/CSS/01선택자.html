CSS 선택자

선택자
특정한 HTML 태그를 선택할 때 사용하는 기능으로, 태그를 선택하여 원하는 스타일과 기능을 적용할 수 있다.

전체 선택자(모든 선택자)

HTML 문서 안의 모든 태그를 선택할 때 사용하는 선택자

[ {color : red; } 와 같은 형태

아이디 선택자

HTML 문서 내에 해당하는 아이디 속성의 값을 가진 태그만 선택할때 사용(id는 중복X)

#IDNMAE {color:red;} 와 같은 형태
  • 아이디 선택자 테스트1
  • 아이디 선택자 테스트2
  • 아이디 선택자 테스트3
  • 아이디 선택자 테스트4
  • 아이디 선택자 테스트5

클래스 선택자

HTML 문사 내 여러개 태그를 동일한 틀래스명으로 지정하여 선택할 때 사용

 .클래스명 {color:red;} 와 같은 형태
  • 클래스 선택자 테스트1
  • 클래스 선택자 테스트2
  • 클래스 선택자 테스트3
  • 클래스 선택자 테스트4
  • 클래스 선택자 테스트5

속성 선택자

기본 선택자 뒤에 []을 이용하여 속성과 속성 값을 사용

선택자[속성=속성값]{color:red;] 형식

후손 선택자 / 자손 선택자

후손선택자 : 하위요소 전부 / 자손선택자 : 바로 아래 하위요소 

 후손 : 선택자 후손선택자{color}/ 자손선택자 : 선택자 >자손선택자{color:red;}

동위 선택자

뒤에 모든태그 선택 : 선택지a ~ 선택지b{color:red;}
