작성된 html 태그 접근

엘리먼트 객체 가져오기

getElement / getElements(배열)

1. id 속성 이용: document.getElemenetById("아이디명")

태그의 아읻속성을 이용해서 태그 정보를 가져오며, id속성은 태그의 고유식별속성으로 동일한 아이디를 여러개 태그에 부여해도 1개의 요소만 가져온다.

2. class 속성 이용: document.getElementsByClassName("클라스명")

클래스 속성으로 태그정보를 가져오며, 동일한 클래스 속성값을 가진 엘리먼트를 모두 가져옴 -> 배열 형태
단, 해당클래스를 사용하는 속성이 1개여도 무조건 배열

3. name 속성 이용: document.getElementsByName("name속성값")

name속성으로 태그정보를 가져오며, 동일한 name속성값을 가진 엘리먼트를 모두 가졍옴 -> 배열 형태
단, 해당 name속성을 사용하는 엘리먼트가 1개여도 무조건 배열

4. 태그 사용 : document.getElemetsByTagName("태그명")

태그 이름을 이용해서 태그정보를 가져오며, 동일한 태그 엘리먼트를 모두 가져옴 -> 배열 형태
단, 해당 태그가 1개여도 무조건 배열

엘리먼트 정보 사용

1. innerHTML 속상 : 엘리먼트 객체의 시작태그와 종료태그 사이의 내용을 문자열로 가져옴

엘리먼트 내부 내용을 읽어올 수 있고 변경도 가능

2. outerHTML 속상 : 시작태그 + 엘리먼트 객체의 시작태그와 종료태그 사이의 내용 + 종료태그 포함하여 문자열로 가져옴

엘리먼트 내용을 읽어올 수 있음! 수정 및 변경 X

3. value 속성 : 엘리먼트 객페의 value 속성 값(주로 input 태그에 사용)

엘리먼트의 value 값을 읽어올 수 있으며 변경 가능
