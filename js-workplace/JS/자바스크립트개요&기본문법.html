자바스크립스 개요

유연한 혹은 체계가 안잡힌 언어

자바스크립트 작성 방식

1. inline : 자바스크립스 양이 한 줄 또는 두줄 정도로 소량일때 사용

2. internal 방식 : html파일 내 자바스크립트 소스를 작성(script 태그 내부 작성)

3. external 방식 : 자바스크립스 파일을 따로 외부 파일로 작성한 후 불러오는 방식

데이터 출력 방법

1. document.write (내용) : 브라우저 화면에 값을 출력

2. window.alert(내용) : 내용을 메서지 창에 출력 (window는 생략 가능하여 alert으로 사용 가능)

3. console.log(내용) : 개발자도구 화면의 콘솔에 출력(개발 도중 값 확인을 위한 용도로 사용)

데이터 입력 방법

1. window.confirm(질문) : 질문에 대해 예/아니오의 결과(true/false)를 얻을때 사용

2. windwo.prompt(질문) : 질문에 대해 입력할 텍스트 필드와 확인/취소 버튼이 있는 대화창 출력

자바스크립스 자료형

자바스크립트는 변수 선언시 자료형을 작성하지 않음 무조건 var

1. 문자열: "",''로 묶여있는 리터럴(자바와 다르게 '',"" 구분없이 모두 문자열로 처리

string이라는 내장 객체로 기본적인 함수(메소드)들이 존재

2. 숫자 : 정수형숫자와 부동소수점 숫자 처리하는 자료형

Math라는 내장객체를 이용하여 기본적인 메소드 사용이 가능

3. 기타자료형: 논리형(true/false), 객체(object), undefined, 함수(function)

자바스크립스 연산자

1. 대부분 연산자는 자바 연산자와 동일

2. ==, != : 데이터만 확인

("1"==1 true)

2. ===, !== : 자료형과 데이터가 모두 일치하는지 확인하는 연산자

("1"===1 false)

3. typeof() : 해당하는 값이 어떤 자료형인지 확인하는 연산자

자바스크립스 형변환

문자열 -> 숫자 형변환 : Number(문자) / 숫자 -> 문자열 형변환 : String(숫자)

자바스크립스 변수

지역변수 & 전역변수

자바스크립스 제어문

1. 조건문(if, if-else, if-else, if, 짧은 조건문(||, &&), : 짧은 조건문을 제외하면 자바와 동일

2. 반복문(for, while): 자바와 동일

2. 분기문(continue, break): 자바와 동일
