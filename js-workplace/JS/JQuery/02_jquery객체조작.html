3. attr(), removeAttr() : 엘리먼트에 속성값을 확인 및 추가/삭제하는 메소드

유사한 기능으로 prop(), removerProp()도 존재(disabled, selected, checked 등에 사용)

3. css() : 선택된 요소의 css속성값을 가져오거나 설정하는 메소드


css 매소드에 매개변수로 css 속성을적으면 해당 값을 확인 가능!

css 매소드에 매개변수 2개 ("속성값", "속성명") css 변경 가능

css를 한번에 여러개 변경하는 경우 1) css().css()
css를 한번에 여러개 변경하는 경우 2) 객체 사용 css({"속성":"속성값"}

5. html(), text(), val()

html() : 선택된 요소의 컨텐츠영역(innerHTML)을 가져오거나 수정

text() : 선택된 요소의 컨텐츠영역(innerHTML)을 가져오거나 수정하지만 가져온 내용 중 태그가 있으면 태그는 가져오지 않음

val() : 선택된 요소의 value를 가져오거나 수정
