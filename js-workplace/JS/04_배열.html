자바스크립트 배열

배열 : 다양한 타입의 데이터를 보관하는 변수 모음

자료향 지정이 없어 모든자료형(숫자,문자열,함수,논리,undefined,객체, 배열)이 저장가능

배열의 선언

배열 함수 : 배열 또한 자바스크립트 객체로 기본적으로 사용가능한 함수들이 존재

indexOf(값) : 배열에서 해당하는 값이 위치한 인덱스 리턴

concat(배열명) : 배열에서 해당하는 값이 위치한 인덱스 리턴

join("구분기호") : 배열을 구분기호로 결합하여 하나의 문자열로 변환해 리턴

reverse() : 배열 순전 반전

push(값) : 배열의 맨 뒤에 요소를 추가(배열 길이가 1증가)

pop() : 배열의 맨 뒤에 요소 제거

unshift() : 배열의 맨 앞에 요소 추가

shift() : 배열의 맨 앞에 요소 제거

slice(idx, idx) : 배열에서 지정한 범위의 값을 복사하여 리턴(원본 배열에 영향X)

splice(idx, howMany val) : 배열에서 index 위피에 요소를 제거/추가/수정(원본배열의 데이터를 변경)
